# Sign Language Translator

Visit the website: [signtotext.aadesh.site](https://signtotext.aadesh.site)

## Running the Code Locally

- **Suggested Python Version**: 3.11.5
- **Install Dependencies**: 
  ```bash
  pip install -r requirements.txt
  ```
- **To Run the Code**: 
  ```bash
  python server.py
  ```
